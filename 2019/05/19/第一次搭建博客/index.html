个人博客搭建过程

第一次尝试搭建个人博客，同时这也是我的第一条博客，以后会在这里记录生活和技术，其他的看心情吧 -。-

博客框架有很多，hexo，typecho，wordpress… 我选择的是hexo，原因是被他的极简风格所吸引，当然，你也可以选择其他框架，只要能找到你喜欢的主题就没其他问题啦~

  • 安装Git Bash和Node.js
  • 安装Hexo
  • 将本地hexo部署到Github

3.将本地hexo部署到github

  • 创建Github page

登录你的Github –> New repository –>仓库名就是 你的Github名称.github.io –> Create repository

  • 启动Github page

    点击右边的Setting –> Launch automatic page generator –>Continue to layouts –> 随意选择一个模板ok

    输入网址例如 https://iamwumaixing.github.io 你就可以看到属于自己的page。

  • 托管hexo到Github

    • 打开hexo文件夹下的 _config.yml 并找到deploy属性
